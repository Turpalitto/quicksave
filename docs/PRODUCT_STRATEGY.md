# Product Strategy

QuickSave is a **privacy-first media library for public Instagram content**.

## Positioning

> Save and organize public Instagram media — no login, no cookies, no noise.

We are **not** a “download anything” tool. Users explicitly share or paste links; we resolve public metadata and help them archive, find, export, and optionally watch public profiles at low frequency.

## Target user

- Creators saving reference material they have rights to use
- Users archiving their own public posts
- Privacy-conscious users who prefer self-hosted resolver
- Power users who want collections, export, and organization

## Core loop

Share / paste link → resolve → download → save → organize → find → export

## Differentiators

1. Privacy-first (no account linking)
2. Self-hosted backend as premium feature
3. Media library (not ephemeral downloads)
4. Creator-safe copy and export attribution
5. Transparent diagnostics without tracking
